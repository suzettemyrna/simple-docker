# Simple Docker

> English version: [README.md](README.md)

Практический проект по изучению контейнеризации с помощью Docker.

В ходе работы последовательно рассматриваются основные возможности Docker: работа с готовыми образами и контейнерами, изменение конфигурации сервисов внутри контейнера, создание собственного Docker-образа, анализ безопасности образов и развёртывание многоконтейнерного приложения с помощью Docker Compose.

В качестве итогового результата создаётся собственный Docker-образ с веб-сервером на FastCGI и обратным прокси на Nginx.

## Темы проекта

* Docker images
* Docker containers
* Docker CLI
* Nginx
* FastCGI
* Dockerfile
* Docker volumes
* Docker networking
* Dockle
* Docker Compose

## Результаты обучения

После выполнения проекта пользователь сможет:

* работать с Docker-образами и контейнерами;
* запускать и останавливать контейнеры;
* анализировать параметры контейнера через Docker CLI;
* изменять конфигурацию сервисов внутри контейнера;
* экспортировать и импортировать контейнеры;
* создавать собственные Docker-образы;
* собирать приложения внутри контейнера;
* настраивать обратное проксирование через Nginx;
* проверять безопасность Docker-образов;
* развёртывать многоконтейнерные приложения с помощью Docker Compose.
